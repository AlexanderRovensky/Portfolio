#  Построение дашбордов в Tableau


## Данные

В наличии были следующие данные:  
* первичный ключ,
* тема карточки,
* тема источника,
* возрастной сегмент,
* дата и время,
* количество событий.
## Задача

Визуализировать взаимодействия пользователей с карточками статей Дзена. 

## Используемые библиотеки
*Pandas, Plotly, Python, Seaborn*
